# hospital-management
Hospital Management System

This is a simple Hospital Management System developed using Python and Flask.

## Features
- Patient Registration
- Store patient details
- View patient records
- SQLite database

## Technologies Used
- Python
- Flask
- SQLite
- HTML
- CSS

## How to Run This Project

### 1. Clone the repository
git clone https://github.com/Velmurugan81/hospital-management.git

### 2. Go to project folder
cd hospital-management

### 3. Install Flask
pip install flask

### 4. Run the project
python app.py

### 5. Open in browser
http://127.0.0.1:5000/
